# Global_pathplanning
基于matlab gui制作的全局路径规划仿真界面，可以使用鼠标点击栅格绘制地图，包括多种障碍物，设置面板可以设定多种数据，内置多种全局路径规划算法，包括多目标a星，多目标ga，多目标蚁群算法，并且内置dfs深度优先算法用以搜索全部可行路径
